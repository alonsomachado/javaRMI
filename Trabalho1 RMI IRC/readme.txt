Para executar o cliente e o servidor é necessário os seguintes passos:

1º javac *.java
2º java −Djava.security.policy=policy.all MensagemServiceServer

Novo terminal para cliente: 

1º java −Djava.security.policy=policy.all MensagemServiceClient

Nota:   Para chat de Grupo deverá entrar primeiro no Grupo pretendido para receber a mensagem dele.

Nota2: Arquivos podem ser enviados para Utilizadores Offline
